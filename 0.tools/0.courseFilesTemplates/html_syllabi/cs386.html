CS 386 Systems Architecture

 

Credit Hours: 4

Course Prerequisites: IS242: Management Information Systems

 

Course Description

This course provides technical knowledge of computer hardware and system software. The material covered in the course presents the background needed for systems analysis, design, configuration, procurement, and management.

 

Course Objectives

The main objectives of this course are to extend the students knowledge of computer systems architectures. At the successful completion of this course the student will be able to:

        Describe system administration responsibilities and tasks

        Compare the functions and layers of an operating system

        Evaluate the distinguishing characteristics of primary and secondary storage

       Compare the logical and physical network topologies including the Internet architecture and network protocol

        Determine the roles, functions, and layers of application and system software

        Evaluate the application development process and the role of methodologies, models, and integrated tools

        Analyze cloud computing models from technical, economic, and security perspectives 

        Analyze centralized and distributed software architectures and Internet protocols for accessing distributed resources 

        Recommend tools and processes for evaluating application resource requirements and computer system performance

        Analyze measures for ensuring system security and threats
