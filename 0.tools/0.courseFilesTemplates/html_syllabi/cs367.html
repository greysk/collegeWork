Programming Languages

Required Text

Sebesta, R.W. (2015). Concepts of Programming Languages, 11th Edition. New York: Prentice Hall.
ISBN: 9780133943061

Course Description

This course provides the tools necessary for the critical evaluation of existing and future programming languages and constructs. It also introduces compiler design and construction.

Course Objectives

Upon completing this course you will be able to:

·         Compare and contrast the different types of programming languages.

·         Identify the tradeoffs, advantages, disadvantages, and design issues inherent in the different programming languages.

·         Analyze the relationship between languages and their capacity to express ideas about computation.

·         Recognize and reduce bugs by understanding programming language syntax and semantics.
