Course Description

Smartphones are powerful handheld devices that combine the functions of a mobile phone and a personal computer. Smartphones contain computer operating systems that support computing tasks. Android and iOS are popular smartphone operating systems found on Android phones and iPhones, respectively. Mobile Applications Development builds applications for iOS and Android platforms. These platforms are incompatible and require separate development environments to create native applications. Separate development for Android and iOS platforms is time-consuming, complex, and costly. This course teaches developers to build mobile applications using cross-platform development methods consisting of HTML, CSS, Ionic, Apache Cordova and Angular.

Course Objectives

         Analyze mobile application requirements and determine the most appropriate mobile development method

         Assemble software components required for a mobile development environment

         Employ AngularJS and Ionic for mobile development

         Design a mobile application using advanced phone features

         Create a mobile application using cross-platform development tools

         Assess the quality and adherence to the design of a mobile application

         Integrate a mobile application on an Android or iOS system, or both systems

         Explain the marketplaces and processes for deploying an application
