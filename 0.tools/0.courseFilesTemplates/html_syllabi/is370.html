Course Description

This course covers methods to build a feature-rich, data-driven, interactive web site using full-stack development approach based in ASP.NET Core MVC.

Course Objectives

Upon completing this course, you will be able to:

  • Create a website using Visual Studio 2017 and Angular 5 to create a single page application (SPA) or Progressive Web Application (PWA) using appropriate tools and file types.
  • Use CSS and CSS Less to add style to webpages.
  • Use Server Controls to add functionality to webpages.
  • Write programs for ASP.NET Core MVC driven webpages.
  • Design and implement a website navigation system.
  • Use Visual C#, JSON, TypeScript, Node.js, Entity Framework Core, MySQL, AJAX, jQuery, and LINQ to interact with a database.
