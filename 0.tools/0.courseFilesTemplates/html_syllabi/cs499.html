Computer Science Capstone Project

Required Text

This course does not use a textbook

Course Description

This capstone course requires demonstration of the knowledge and skills gained throughout the Computer Science degree program by designing and implementing a software program or computer-related system to solve a real-world problem. The project requires project definition, requirements determination, design, implementation, test, and documentation of the system.

Course Objectives

·         Apply knowledge of computing and mathematical reasoning related to computer science

·         Define the computing requirements appropriate to its solution

·         Justify the necessity of the software solution through a detailed requirements analysis.

·         Develop a design methodology and business and technical case for the software solution.

·         Translate a design methodology into a physical software application.

·         Construct a proof of concept

·         Evaluate the effectiveness of a software application

·         Test a physical application

·         Refine application based on recommended areas of improvement or optimization for the software solution.

·         Compose a final written solution report

·         Demonstrate the use of a physical application

·         Communicate effectively with a range of audiences

Credit Hours: 3

Prerequisites: Completion of degree Requirements
