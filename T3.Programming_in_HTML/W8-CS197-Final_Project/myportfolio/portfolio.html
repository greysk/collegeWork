colorvalue

Use colorvalue to obtain the RGB and HEX values of HTML color names or to obtain HTML color names by color group. Optionally, colorvalue can send the returned values directly to the clipboard.

pfix

Use pfix to quickly change the path separators in a file path from Windows-style (\) to UNIX-style (/).
